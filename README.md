# JavaScript-regex

常用的正则表达式总结,见regex.js

#### 验证数字
```
    var reg = /^[0-9]*$/;<br>
    var texts = "12132";<br>
    if(reg.test(texts)){
      console.log("验证数字");
    }
```

### 贴几个自己常用到的（拿来即用）

1. `/^(\d+)(\.?)(\d{0,2})$/`   // 输入整数或小数，保留2位小数
2. `/^[0-9.%]{1,}$/`  // 输入百分数值
3. `/^1[34578]\d{9}$/`  // 验证为手机号码
